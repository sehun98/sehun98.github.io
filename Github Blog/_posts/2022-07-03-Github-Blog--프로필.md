# Github Blog 프로필

`_config.yaml` 에서 프로필을 입력합니다.

## # Site Settings

```yaml
locale                   : "ko-KR"
title                    : "정리하는 개발자 Blog"
title_separator          : "|"
subtitle                 : "Version 1.0" 
name                     : "오세훈(Oh sehun)"
description              : "OOOOOO는 공부 습관"
url                      : "https://sehun98.github.io" 
baseurl                  : # the subpath of your site, e.g. "/blog"
repository               : "https://github.com/sehun98/sehun98.github.io"
logo                     : "/assets/images/88x88.png"
masthead_title           : "OOOO는 개발자"
```

author:
  name             : "Oh sehun"

  bio              : "**기록이 자산!!**<br>매일 공부하고 있습니다.<br>"
  location         : "Seoul, Republic of Korea"

  links:

      # url: "https://your-website.com"
