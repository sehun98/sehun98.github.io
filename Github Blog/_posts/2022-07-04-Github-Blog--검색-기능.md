# 우측 상단에 검색 기능 활성화

`_config.yaml` 에서 검색 기능을 활성화 합니다..

## # Site Settings

```yaml
search                   : true # true, false (default)
search_full_content      : # true, false (default)
search_provider          : # lunr (default), algolia, google
lunr:
  search_within_pages    : true # true, false (default)
```
