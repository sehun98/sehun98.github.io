# ⭐ 2’s complement ⭐

Decimal의 경우 `±`와 `값의 크기`를 통해 숫자를 표현하지만 컴퓨터의 경우 그러한 연산이 불가능 합니다.

따라서 컴퓨터에서 숫자의 표현이 필요했고, 2’s complement, IEEE 754 format을 채택하였습니다.

------

`2's complement`  1's complement -> +1

- 0001
- 1110 // 1’s complement
- 1111 // 1’s complement +1 // 2’s complement
