### SMD 정격 결정방법

전압 5V일때 가정합니다.

저항의 경우 사이즈만 결정해도 되지만 캐패시터의 경우 전압을 함께 고려해야 합니다.

필요한 전력의 1.5배로 가정하고 정격 전력마진을 고려해야 합니다.

| 사이즈     | 정격전력 [W] | 정격 전류 [A] |
| ---------- | ------------ | ------------- |
| 0201(0603) | 1/20         | 0.01          |
| 0402(1005) | 1/16         | 0.0125        |
| 0603(1608) | 1/16         | 0.0125        |
| 0805(2012) | 1/10         | 0.02          |
| 1206(3216) | 1/8          | 0.025         |
| 1210(3226) | 1/4          | 0.05          |
| 2020(5050) | 1/2          | 0.1           |
| 2512(6330) | 1            | 0.2           |

| 허용오차 | 등급 |
| -------- | ---- |
| ±0.1%    | B급  |
| ±0.25%   | C급  |
| ±0.5%    | D급  |
| ±1%      | F급  |
| ±2%      | G급  |
| ±5%      | J급  |
| ±10%     | K급  |
| ±20%     | M급  |

보통 F, G, J, K을 사용합니다.
