# Open Git bash here 

#### Git을 설치 할때 Open Git bash here을 체크 해줘야 사용가능한 기능입니다.

> 이미 Git을 설치했다면, 업데이트를 진행합니다. 명령 프롬프트(`cmd`) 혹은 `git bash`에서 아래와 같이 입력합니다.

```bash
$ git update-git-for-windows
```
