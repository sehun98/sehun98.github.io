`umm` : 단위 설정

`s 0 0` : 커서 위치 변경

`gd0.1` : 그리드 변경

`g0.1` : 그리드 변경

`w0.127` : 굵기

`ad` : 각도 변경

`z1` : 레이어 변경

`ctrl` + `e` : 부품 이동

`ctrl` + `r` : 부품 회전

`ctrl` + `alt` + `c` : 색상 변경

`F2` : 선 연결

`drp` : 에러 체크

`F2` > `Shift` + `Space` > `tap` : POWER

`F2` > `Ctrl` + `Space` : GND

`Alt` + `Space` > `Ctrl` + `R` : CON
